# Bosai Architect by Tatarat

防災教育コーディネーター・支援エージェント設計ツール

## 概要

自治体・学校・施設の防災教育・支援担当者向けに、AIによる防災教育プログラムと支援エージェントを設計するインタラクティブツール。

- **防災対象 8種**（地震・津波・火災・水害・土砂・台風・感染症・複合）
- **ターゲット 7カテゴリ**（児童・青年・組織・高齢者・外国人・障がい者・地域横断）
- **5層構造の40ブロック**による組み合わせ設計
- **15のテンプレアーキテクチャ**（ミニマム版/フル統合版の2案提示）

## 技術構成

- **単一HTMLファイル**（外部依存：Google Fonts のみ）
- **クライアントサイドのみ**（バックエンド不要、INSTANT即時応答モード）
- **モバイル対応**（iPhone / Android Safari でフル機能動作）

## デプロイ

Cloudflare Pages による静的ホスティング。

- Public URL: `https://bosai-architect.pages.dev/`
- Build command: なし
- Output directory: `/`

## ローカルプレビュー

```bash
# 静的サーバーを立てるだけ
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

または `index.html` をブラウザで直接開く（Safari/Chrome推奨）。
